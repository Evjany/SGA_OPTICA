## COMPONENTE METODOLÓGICO
## Nombre del proyecto : 
S.G.A OPTICA - IMPLEMENTACION DE UN SISTEMA DE GESTION DE CITAS ONLINE PARA OPTICA

## Planteamiento del problema 
La óptica Unidad Visual enfrenta dificultades porque no dispone de un sistema digital para gestionar sus citas. Actualmente, los usuarios deben reservar de forma presencial o por teléfono, un proceso que depende de la disponibilidad del personal y que genera errores, pérdida de información, desorden en la agenda y tiempos de espera prolongados. Esta situación afecta directamente la experiencia del usuario, ya que no hay confirmaciones automáticas, no existen recordatorios y no se puede consultar la disponibilidad del optometrista en tiempo real.
En un entorno donde la digitalización es una necesidad para la competitividad, mantener procesos manuales coloca a la óptica en desventaja frente a otras que ya emplean sistemas modernos y automatizados.

## Pregunta problema
¿Cómo implementar un sistema de información web que permita gestionar citas, ventas de productos ópticos y consultas administrativas internas de forma organizada, automatizada y accesible tanto para los usuarios como para el personal de la óptica Unidad Visual?

## Objetivo general
Diseñar e implementar un sistema de información web y una aplicación móvil para optimizar procesos. El componente web centraliza las consultas administrativas, el control de inventario, la gestión operativa interna y la programación de citas optométricas. Por su parte, la aplicación móvil optimiza la experiencia del usuario final al permitir la compra ágil de productos ópticos, mejorando así la eficiencia operativa y la fidelización del cliente.
## Objetivos especí­ficos
1. Desarrollar el módulo de citas, con su interfaz, tablas asociadas y pruebas funcionales.
2. Implementar el módulo de ventas que incluya catálogo digital, carrito de compras, pagos en línea y pruebas correspondientes.
3. Construir el módulo administrativo para consulta de inventario, gestión de productos y visualización de citas.
4. Integrar los módulos en una base de datos centralizada que garantice seguridad, consistencia y disponibilidad de la información
5. Realizar pruebas funcionales e integrales para validar el correcto funcionamiento de cada módulo del sistema

# Alcance del proyecto 
Agendamiento de citas con disponibilidad en tiempo real. Envío automático de recordatorios y confirmaciones. Gestión de un catálogo digital de productos. Registro de compras, procesamiento de pagos y gestión de pedidos. Consulta y actualización de inventario para uso administrativo. Roles diferenciados para cliente, empleado y administrador.
## Requisitos funcionales:
Agendamiento de citas con disponibilidad en tiempo real.
Envío automático de recordatorios y confirmaciones.
Gestión de un catálogo digital de productos.
Registro de compras, procesamiento de pagos y gestión de pedidos.
Consulta y actualización de inventario para uso administrativo.
Roles diferenciados para cliente, empleado y administrador.

## Requisitos no funcionales:
Accesible desde dispositivos móviles y de escritorio.
Interfaz amigable y fácil de usar.
Seguridad en el manejo de datos.
Base de datos robusta, unificada y consistente.
Automatiza la gestión de citas.
Administra ventas de productos ópticos.
Permite registrar inventarios, pedidos y usuarios.
Envía notificaciones automáticas.
Guarda historial completo de usuarios y citas.

## Justificación
El proyecto consiste en desarrollar e implementar un sistema automatizado para la optica Unidad Visual; El proyecto busca solucionar los problemas generados por el sistema manual de agendamiento utilizado por la óptica, el cual provoca errores, demoras, pérdida de información y falta de recordatorios para los usuarios. La implementación de un sistema web permitirá automatizar la gestión de citas, mejorar la organización interna y ofrecer una experiencia más rápida y cómoda.
Además, la integración de una tienda virtual ampliará los servicios de la óptica, facilitando la compra de productos visuales y optimizando procesos de venta e inventario. En conjunto, el sistema fortalecerá la eficiencia operativa, mejorará la atención al cliente y aumentará la competitividad de la empresa.
